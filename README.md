# Hybrid Solar System Design for Ugandan Homes 🇺🇬

## 📌 Overview
This project presents a cost-optimized hybrid solar system designed specifically for Ugandan households facing frequent power outages and high energy costs.

The system prioritizes **critical loads** (lighting, TV, WiFi) to ensure uninterrupted power, while reducing battery and system costs.

---

## ⚡ Problem Statement
In Uganda, many households experience:
- Frequent power outages
- High cost of solar systems
- Expensive and unreliable batteries
- Dependence on generators

---

## 💡 Proposed Solution
A hybrid system integrating:
- Solar panels ☀️
- Battery storage 🔋
- Grid (Yaka) ⚡
- Optional generator ⛽

### Load Strategy:
- ✅ Critical Loads: Always powered (lights, TV, WiFi)
- ⚠️ Non-Critical Loads: Powered only when energy is available

---

## 🧠 Key Innovation
Instead of powering the entire house, this system:
- Guarantees power for essential devices
- Reduces battery size and cost
- Improves affordability for average households
